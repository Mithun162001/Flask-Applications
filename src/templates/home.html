Remote Server File Upload Application

Welcome to the Remote Server File Upload Application


This application allows you to securely upload files to a remote server using the SCP (Secure Copy Protocol) method.



Follow the instructions below:


  1. Enter the IP address of the remote server below and click "Connect."

  2. Enter your username and password to authenticate.

  3. Click the "Upload File" button to transfer a file to the remote server.
